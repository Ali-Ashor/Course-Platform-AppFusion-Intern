# 🎓 Course Selling Platform (Full Stack Project)

A simple and dynamic **Course Selling Platform** built using **Node.js, Express, and SQLite**.
This project allows users to register, login, view courses, and enroll easily.

---

## 🚀 Features

* 👤 User Signup & Login (Authentication)
* 📚 View Available Courses
* 🛒 Enroll in Courses
* 🗂️ SQLite Database Integration (No external setup required)
* 🔐 Session-based login system
* ⚡ Simple and clean UI

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS
* **Backend:** Node.js, Express.js
* **Database:** SQLite
* **Session Management:** express-session

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Ali-Ashor/Course-Platform-AppFusion-Intern.git
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Run the server

```bash
node server.js
```

### 4️⃣ Open in browser

```
http://localhost:3000
```
---

## 📌 How It Works

1. User signs up and logs in
2. After login, user can view available courses
3. User can enroll in any course
4. Data is stored in SQLite database
